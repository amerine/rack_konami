Rack::Konami

A Rack middleware component that embeds the Konami Code in your app and lets you fade in and out a snippet of html.

Requirements

It requires JQuery 1.4.0 or greater.

Installation

gem install rack_konami

Usage

require 'rack_konami'
use Rack::Konami, :html => "<img src='/images/rails.png'>"
app = lambda { |env| [200, { 'Content-Type' => 'text/html' }, '<html><body><p>Awesome Body</p></body></html>'] }
run app

Rails 3.x / Sinatra

In your Gemfile 

gem "rack_konami"

In your config.ru

use Rack::Konami, :html => '<img src="/images/rails.png" >', :delay => '1500'
